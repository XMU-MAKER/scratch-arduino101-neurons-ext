Arduino101 Scratch Extension with machine learning supporting

A easy way to let kids know what is machine learning

支持机器学习功能的Arduino 101 Scratch扩展

本扩展源码编译需要General Vision的CurieNeurons free library支持。若不想编译，请参照下面的说明，使用编译好的firmware

使用方法

烧写firmware

下载已编译的firmware及烧写工具，

解压

如果未安装驱动，请先运行tools\driver_install.bat，安装arduino 101驱动

连接arduino101开发板到电脑，按MASTER REST按键，然后运行upload.bat烧写firmware。

启动scratch并加载扩展

请先安装Scratch Device Plugin - 下载

Sratchx with machine learning supported on arduino 101

Arduino 101 Scratch Extension with machine learning

Please install the CurieNeurons free library of General Vision before compiling the firmware. If you don't want to compile the firmware, please read the following instruction

Usage

Flash firmware

download the firmware & tools.

decompress it.

Install the arduino 101 device driver by run tools\driver_install.bat

Flash the firmware by run upload.bat after connected the arduino 101 to USB and press MAST RESET

Open Scratchx

The extension require the Scratch Device Plugin - Download here
